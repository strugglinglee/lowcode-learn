# lowcode-learn
低代码学习项目

## 项目结构

```
├── src
│   ├── common                // 存放公共组件
│   ├── components            // 存放物料组件
│   ├── contexts              // 存放...
│   ├── layouts               // 布局
│   │   ├── header            // 头
│   │   ├── material          // 物料区
│   │   ├── setting           // 配置区
│   │   └── renderer          // 中间渲染区
│   └── utils                 // 存放工具方法
```

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
